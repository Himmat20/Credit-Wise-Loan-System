# Credit Risk Loan Prediction System

This project focuses on building an intelligent loan approval system using Machine Learning to help financial institutions make faster, more accurate, and unbiased decisions.

## Problem Statement

Traditional loan approval processes rely heavily on manual verification of documents such as income proof, employment details, and credit history. This approach is:

- Time-consuming
- Prone to human bias
- Inconsistent in decision-making

As a result:
- Good customers may get rejected
- High-risk customers may get approved

This project aims to solve these issues using a data-driven approach.

---

## Objective

To build a Machine Learning model that predicts whether a loan application should be:

- Approved
- Rejected

based on applicant details.

---

## Dataset Description

Each row in the dataset represents a loan applicant and includes:

- Personal Information
- Financial Details
- Credit History
- Employment Information

---

## Project Workflow

### 1. Exploratory Data Analysis (EDA)
- Understanding data distribution
- Identifying missing values
- Correlation analysis using heatmap

### 2. Data Preprocessing
- Handling missing values
- Feature Encoding (Categorical → Numerical)
- Feature Scaling using StandardScaler

### 3. Model Building

The following algorithms were implemented:

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

### 4. Model Evaluation

Models were evaluated using:

- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix

---

## Results

The models were able to identify patterns in customer data and predict loan approval decisions effectively.
